# JDownloader

JDownloader is a Java based download manager for Windows, Linux and Mac OS X. It is based on the open source project JDownloader2.

## Usage

```bash
skaffold dev
```

## Port Forwarding

```bash
skaffold port-forward
```

## Accessing JDownloader

JDownloader is accessible at http://localhost:5800
